# Curso: Introduction to JavaScript (TAU) 🧪

Este repositório foi criado para armazenar meus estudos, anotações e códigos práticos desenvolvidos durante o curso **Introduction to JavaScript** da trilha JavaScript Web UI da *Test Automation University (TAU)*. 

O objetivo principal deste treinamento é consolidar uma base em lógica de programação e na sintaxe da linguagem JavaScript, preparando o terreno para o desenvolvimento de testes automatizados (como Cypress, Playwright ou API Testing).

---

## 🎯 Objetivo dos Estudos
Este repositório contém minha jornada na aprendizagem da sintaxe JavaScript com o objetivo de me preparar para produzir testes automatizados em frameworks como Cypress, Playwright e, futuramente, em APIs.
---

## 📂 Estrutura do Projeto

Atualmente, o repositório contém os fundamentos iniciais da linguagem, divididos nos seguintes arquivos:

*   **`01-variables.html`**: Estudo sobre declaração de variáveis (`var`, `let`, `const`) e escopo de memória.
*   **`02-data-types.html`**: Manipulação e identificação dos tipos de dados primitivos (Strings, Numbers, Booleans, Null e Undefined).
*   **`03-arrays.html`**: Estruturas de dados em formato de listas, criação de arrays e métodos básicos de manipulação.

---
